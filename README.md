# Codex10k Revenue Run

Independent revenue-run workspace for reaching USD 10,000 in verified net profit.

This run is separate from any prior `Revenue Run` Gmail labels, repos, demos, or outreach. New Gmail labels use the `Codex10k/` namespace.

## Offer

Primary offer: **10-Day Workflow Revenue Recovery Sprint**

- Fixed-scope implementation for one revenue or operations bottleneck.
- Typical targets: missed lead response, quote intake, CRM cleanup, internal reporting, and AI-assisted inbox/support triage.
- Demo-first sales motion: show a working prototype, then sell a paid pilot.
- No unverifiable credential claims, fake case studies, fake reviews, or deceptive outreach.

## Profit Definition

Verified net profit counts only settled revenue minus direct costs, platform fees, payment fees, refunds, ad spend, subscriptions, and purchased assets.

Ledger: `data/ledger.csv`
