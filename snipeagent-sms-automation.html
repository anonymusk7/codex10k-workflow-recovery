Suggested workflow

Turn the scope into staged, testable releases

The safest path is to confirm the architecture and edge-case plan before wiring every scenario. That keeps the NDA-protected full scope useful without turning the first week into guesswork.

01 Inbound capture Sinch SMS/MMS webhook, request ID, sender normalization, and message-state record.
02 MMS handling Download media, encode safely, pass image context to AI, and log failures.
03 Processing lock Prevent overlapping requests for the same user and recover stale locks.
04 Parallel search Run Rainforest and SerpApi branches with timeouts, retries, and result collation.
05 AI formatting Claude Haiku returns structured result copy, fallbacks, and safe reply text.
06 Reply and re-engage Send SMS, update analytics/savings stores, and queue approved re-engagement rules.
